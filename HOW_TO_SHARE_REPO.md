## üë• Method 2: Add Collaborators (For Private Repos)

If your repository is **private**, add specific people:

### Steps:
1. **Go to your repository**: https://github.com/ORZaid/CDPTA-HTA
2. **Click "Settings"** (top menu bar)
3. **Click "Collaborators"** (left sidebar)
4. **Click "Add people"** button
5. **Enter GitHub username or email** of the person
6. **Choose permission level:**
   - **Read**: Can view and clone
   - **Write**: Can push changes
   - **Admin**: Full control
7. **Click "Add [username] to this repository"**
8. **They'll receive an email invitation** to accept

### Permission Levels Explained:
- **Read**: View, clone, download code
- **Write**: Everything in Read + push changes, create branches
- **Admin**: Everything in Write + manage settings, delete repo
